# UI preference — owner-approved layout

## Simple sidebar (default — 8 items)

Property owners see **only** these in the left nav:

| # | Label | Page id |
|---|-------|---------|
| 1 | Dashboard | `dashboard` |
| 2 | Properties | `buildings` |
| 3 | Units | `units` |
| 4 | Tenants | `tenants` |
| 5 | Payments | `payments` |
| 6 | Deposits | `deposits` |
| 7 | Reports | `reports` |
| 8 | Settings | `settings` |

## Advanced tools (Settings → More tools)

Everything else stays accessible but **not** in the sidebar:

- Plans & Billing (`subscription`)
- Data Import (`data-import`)
- Guided Steps (`guided`)
- Ask Assistant (`assistant`)
- Vacancy Board, Lease Manager, Rent Ledger, Utilities, Messages, Maintenance
- Documents, Legal Notices, Blacklist, Defaulters, Help

## Login copy

- Heading: **Welcome back**
- Subtext: Sign in to manage your properties (Uganda landlords)

## Do not restore

- Long sidebar with 20+ items
- Instant subscription activate on MoMo ref
- Kenya branding copy
